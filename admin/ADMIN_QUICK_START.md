## 🎯 Common Tasks

### 1. Adding a New Robot

1. Click the **"+ Add New Robot"** button at the top
2. Fill in the form sections (marked with * are required):
   - **Basic Information**: Name, Type, Category
   - **Upload Image**: Drag & drop or click to select
   - **Description**: Short overview and detailed info
3. Click **"✅ Create Item"** at the bottom
4. You'll see a green success message!

**Time**: ~2-3 minutes

### 2. Adding a Gallery Image

1. Click **"📸 Gallery"** tab at the top
2. Click **"+ Add New Image"** button
3. Fill in:
   - **Title** and **Category**
   - **Description** (keep it under 200 characters)
   - **Event details** (location, date, participants)
   - **Main image** (drag & drop supported!)
   - **Additional images** (optional, multiple allowed)
4. Click **"✅ Create Item"**
5. Success! Image appears in the gallery

**Time**: ~3-5 minutes

### 3. Editing Existing Items

1. Find the item you want to edit
2. Click the **"✏️ Edit"** button on the item card
3. Make your changes
4. Click **"💾 Update Item"**
5. Done! Changes are saved

**Time**: ~1-2 minutes

### 4. Deleting Items

1. Find the item to delete
2. Click the **"🗑️ Delete"** button
3. Confirm when asked "Are you sure?"
4. Item is removed immediately

**Time**: ~10 seconds

### 5. Filtering by Category

1. Use the filter buttons below the stats:
   - **Robots**: All, Competition, Research, Development
   - **Gallery**: All, Robots, Events, Workshops, etc.
2. Click any category to see only those items
3. Click **"All"** to see everything again

**Time**: ~5 seconds

## 💡 Tips & Tricks

### Image Upload

✅ **DO**:
- Use high-quality images (but under 5MB)
- Supported formats: JPG, PNG, GIF, WebP
- Drag & drop is faster than clicking!
- Preview images before submitting

❌ **DON'T**:
- Upload images over 5MB
- Use corrupt or invalid image files
- Forget to check the preview

### Form Filling

✅ **DO**:
- Fill required fields (marked with red *)
- Read helper text under inputs
- Use comma-separated lists for tags
- Check character counters

❌ **DON'T**:
- Skip required fields
- Paste very long text without checking limits
- Submit without previewing

### Best Practices

1. **Name Things Clearly**
   - ✅ "LineBot Pro - Competition Winner 2024"
   - ❌ "robot1" or "IMG_001"

2. **Write Good Descriptions**
   - ✅ "Autonomous line-following robot with adaptive speed control"
   - ❌ "A robot that follows lines"

3. **Choose Correct Categories**
   - Competition robots → 🏆 Competition
   - Research projects → 🔬 Research
   - Workshop photos → 🛠️ Workshops

4. **Use Multiple Images**
   - Add 3-5 images for gallery events
   - Show different angles for robots
   - Include action shots

## ⚠️ Troubleshooting

### "Please upload an image"
**Solution**: Make sure you've selected an image file before submitting

### "Image size should be less than 5MB"
**Solution**: Use a smaller image or compress it first (use tinypng.com)

### "Please select a valid image file"
**Solution**: Only use JPG, PNG, GIF, or WebP formats

### "Failed to save"
**Solution**: 
- Check your internet connection
- Make sure all required (*) fields are filled
- Try again in a few seconds

### Can't see my new item
**Solution**:
- Check if you're on the right tab (Robots vs Gallery)
- Check the category filter
- Refresh the page

### Form disappeared after I started filling it
**Solution**:
- Click "Add New" again
- Your last submission might have succeeded - check the items list

## 📞 Need Help?

### Common Questions

**Q: How do I know if something saved?**
A: You'll see a green "✅ Success!" message at the bottom right

**Q: Can I undo a delete?**
A: No, deletions are permanent. Always confirm you want to delete.

**Q: How many images can I upload?**
A: One main image + unlimited additional images (gallery only)

**Q: What's the character limit for descriptions?**
A: Short description: 200 characters | Long description: No limit

**Q: Can I edit items after creating them?**
A: Yes! Click the "Edit" button on any item card
